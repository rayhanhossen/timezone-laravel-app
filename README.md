# Time Zone

*It's a simple e-commerce web app . I build this app using php Laravel Framework . For front-end design i used colorlib timezone templete. In the dashboard panel you can add product categories, add product and product images. You can also see customer details and payments. when you add the product in cart its wating for checkout. Then you need to fill up the information and proceed to order*
